# AfekaJbareenTravelSites

אתר פשוט על "5 האתרים הכי יפים שבקרתי או רוצה לבקר".

## מבנה הפרויקט
- `index.html` - דף הבית עם כותרת, תפריט וקישור תמונה למפות.
- `place1.html` - דף על ירושלים.
- `place2.html` - דף על ברצלונה.
- `place3.html` - דף על האלפים השוויצריים.
- `place4.html` - דף על דאהב.
- `place5.html` - דף על איסטנבול.
- `style.css` - עיצוב משותף לכל הדפים.
- `images/` - תיקיית תמונות עם קבצים מקומיים.

## הוראות
1. פתחו את `index.html` בדפדפן.
2. לחצו על קישורים בתפריט או על המפה כדי להיכנס לעמודים של המקומות.
